Message Queue

Amazon SQS

A fully managed message queue that decouples microservices for reliable, scalable communication.

📖 What is Amazon SQS?

Amazon SQS (Simple Queue Service) is a fully managed message queuing service. It lets you send, store, and receive messages between software components — without losing messages or requiring each component to be available at the same time.

Think of it as a waiting line: Lambda 1 drops the order into the queue, and Lambda 2 picks it up when it's ready to process.

⚙️ How Does It Work?
  • Producer — Sends messages to the queue (OrderHandler Lambda)
  • Queue — Stores messages until a consumer processes them
  • Consumer — Polls and processes messages (OrderProcessor Lambda)
  • Visibility Timeout — Prevents duplicate processing
  • Dead Letter Queue — Failed messages are moved here for debugging
💻 Code Example
Python — Send a message to SQS
import boto3, json sqs = boto3.client("sqs", region_name="ap-south-1") # OrderHandler sends the order to the queue sqs.send_message( QueueUrl="https://sqs.ap-south-1.amazonaws.com/123456/OrderQueue", MessageBody=json.dumps({ "orderId": "ORD-A1B2C3", "customerName": "Naman", "item": "Paneer Burger" }) ) # OrderProcessor receives it automatically via event source mapping def lambda_handler(event, context): for record in event["Records"]: order = json.loads(record["body"]) print(f"Processing order: {order['orderId']}")
🍔 How I Used It in QuickBite

📨 OrderQueue

When a customer places an order, Lambda 1 (OrderHandler) saves it to DynamoDB and then sends the order JSON to the OrderQueue. This decouples the fast API response from the slower email-sending process.

🔗 Event Source Mapping

Lambda 2 (OrderProcessor) is configured with an SQS trigger. When a message arrives in the queue, AWS automatically invokes the processor — which sends the SES receipt and SNS alert.

💪 Why SQS?

Without SQS, if the email service was slow, the customer would wait. With SQS, the API returns instantly with the orderId while email processing happens asynchronously in the background.
